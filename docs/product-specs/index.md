# Product Specifications Index

## プロジェクト概要

Switchbot温湿度CO2センサーから値を定期的に取得し、WebUI上でグラフ化するWebアプリケーション。

## 目標

### ユーザー

- **プライマリ**: 自宅の環境データを監視したい個人ユーザー
- **セカンダリ**: 複数の部屋の環境を比較したい人

### ユーザーニーズ

1. **現在の環境状態を一目で確認**
   - 温度、湿度、CO2濃度
   - 快適範囲との比較

2. **時系列での変化を可視化**
   - 1時間、24時間、1週間のグラフ
   - トレンドの把握

3. **アラート（将来）**
   - 閾値を超えた場合の通知
   - 換気が必要なタイミング

## MVP機能（Phase 1）

### 必須機能

- [ ] Switchbot APIからセンサーデータを取得
- [ ] 現在の温度、湿度、CO2濃度を表示
- [ ] 過去24時間のグラフ表示
- [ ] 1分ごとの自動更新
- [ ] ローカルストレージでのデータキャッシング

### 非機能要件

- レスポンシブデザイン（デスクトップ + モバイル）
- ダークモード対応
- 初期ロード3秒以内

## 将来の拡張（Phase 2+）

- [ ] 複数センサーのサポート
- [ ] データの永続化（データベース）
- [ ] カスタマイズ可能なアラート設定
- [ ] データのエクスポート（CSV, JSON）
- [ ] 比較ビュー（複数部屋）
- [ ] WebSocketによるリアルタイム更新
